Comandos para movimentação

  • Ao contrário do editor de texto comum, você utiliza as teclas |h|, |j|, |k| e |l| em vez das setas do teclado para mover o cursor.
  • Exercício: Encontre o erro no código abaixo e o corrija, use as teclas |h|, |j|, |k| e |l| para se movimentar no código.
  • int main() {
    // printf() displays the string inside quotation
    printf("Hello world")
    return 0;
}
